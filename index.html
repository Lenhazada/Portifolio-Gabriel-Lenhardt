Meus Projetos

DsList

Um aplicativo em Java criado para organizar e apresentar listas de jogos de maneira estruturada. O projeto foi estruturado em camadas, interage de forma eficiente entre o back-end e o banco de dados, e segue boas práticas de desenvolvimento, permitindo operações como listagem, ordenação e consulta de dados. Foi criado com foco em aprendizado de APIs REST, organização de código e modelagem de dados.

RentService

Um sistema feito em Java que faz o gerenciamento de aluguel de carros, onde é possível cadastrar locatários, acompanhar a disponibilidade e ter os dados organizados de maneira estruturada. O projeto foi elaborado para reforçar a lógica de programação, o uso de classes, o encapsulamento e a manipulação de dados, solidificando conceitos cruciais do desenvolvimento orientado a objetos.

TaskManager API

Esta API RESTful foi desenvolvida para o gerenciamento de tarefas, permitindo criar, listar, atualizar e remover registros por meio de operações CRUD. A aplicação segue uma arquitetura em camadas, utiliza controladores REST, repositório para acesso a dados e respostas HTTP padronizadas, reforçando conceitos de APIs, persistência e boas práticas no desenvolvimento back-end.
